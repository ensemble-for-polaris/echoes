### Claude Code-First Design Philosophy

**Key insight from user:** "A lot of people use Claude Code" ‚Üí we should support that workflow directly

**Current toolkit assumes:**
```python
# API mode - requires keys
response = client.messages.create(...)
```

**Proposed enhancement:**
```python
# Support both modes
class LLMIntegration:
    def __init__(self, mode="claude-code"):  # or "api"
        ...
    def think(self, context):
        if mode == "claude-code":
            # Use active Claude Code session
        elif mode == "api":
            # Direct API call
```

**Why this matters:**
- No API setup required for Claude Code users
- Immediate usability
- Users see agent reasoning transparently
- Works with whatever model they have access to
